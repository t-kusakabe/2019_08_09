# vim操作編

---

きっとみなさんが一番挫折するポイント。

>>>

とはいえ普通に使う分にはそんなに覚えることはない！

>>>

vimにはいろいろModeがある

>>>


`i` と `esc` で入力とコマンドModeを切り替える

>>>

移動は `h`, `j`,`k`,`l`

>>>

保存は `:w`

>>>

これだけ覚えておけば今日からでもvimで作業ができます！

# vim plugin編

>>>

vimのいいところはカスタマイズ出来るところ。

>>>

まずはpluginを管理しやすい基盤を作る。

>>>

- Vundle
- vim-plug
- NeoBundle
- dein.vim

>>>

今回はvim-plugを使います。

>>>

```
call plug#begin()

# 入れたいplugin
Plug 'cocopon/iceberg.vim'

call plug#end()
```

>>>

たったこれだけ！

>>>

あとは好きなpluginを書いていくだけです。

>>>

ちなみに僕の構成はこんな感じです。

```
.
├── colors
│   └── iceberg.vim
├── ftplugin
│   ├── dart.vim
│   ├── go.vim
│   ├── php.vim
│   ├── pug.vim
│   ├── ruby.vim
│   └── yaml.vim
├── init.vim
└── userautoload
    ├── basic.vim
    ├── plug.vim
    └── statusline.vim
```
